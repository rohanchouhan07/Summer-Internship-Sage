# internal css
Internal CSS is a method of applying styles to a specific HTML document by including the CSS code within the <style> tags in the head section of the HTML file. This allows you to define styles that will only apply to that particular document, making it easier to manage and maintain your styles for that specific page.

<style>
  h1 {
    color: green;
    font-size: 30px;
  }
  
  p {
    color: orange;
    font-size: 18px;
  }
</style>
<body>
<h1>Internal CSS Example</h1>
<p>This is a paragraph styled with internal CSS.</p>
</body>

#external css
External CSS is a method of applying styles to multiple HTML documents by linking an external CSS file to the HTML files. This allows you to maintain a single CSS file that can be reused across multiple pages, making it easier to manage and update your styles in one place.
<!-- Link to external CSS file -->
<link rel="stylesheet" type="text/css" href="styles.css">
